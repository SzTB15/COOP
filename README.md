A végén kiírja a megmaradt pénzünket, ha többe kerülnek a vásárolt termékek akkor kiírja, hogy nincs elég pénz.

termekek.csv: Itt lesznek a termékek.
          tej;469
          cukor(1kg);300

          pick felvágott(/kg);2500
          coop felvágott(/kg);1800

          olaj(1kg)
